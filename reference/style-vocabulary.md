# Style vocabulary — exemplar seeds for real divergence

Read this when running phase B, especially Mode B round 1 (archetype) or any style-flavored axis.

**Why this file exists:** when you name archetypes spontaneously, you sample your own mode — the same 5–6 high-frequency styles every time. The option space of actual shipped design languages is an order of magnitude larger. This file is the offline floor of that space (distilled from public design-language taxonomies and style libraries); it is a menu to *pull cards away from the center*, not a complete atlas.

## How to use (the exemplar mechanic)

1. When mapping the option-space (diversity-recipe §3.3), pull candidates from the families below **plus** anything the subject's own world suggests (its materials, instruments, vernacular — often the most distinctive seed).
2. Assign each constraint card **ONE named language + its signature line** to metabolize. **Different exemplar per card, never shared** — a shared reference anchors all cards and collapses diversity. On the archetype axis, cards should come from **different families**.
3. An exemplar is a *lens*, not a skin: the card interprets the axis through that language while still using the locked tokens and real data verbatim. If an exemplar's palette fights the brand tokens, keep the tokens and take the language's *structure* (grid, density, ornament logic, type attitude).
4. **Known-mode cap: at most ONE card per round from the list below.** Those looks are legitimate, but they are the distribution's center — the model produces them unprompted, so spending 2+ cards there buys no information.
5. This file is the floor, not the ceiling: if a richer style library is installed (e.g. a styles catalog skill), mine it for more candidates; with web access, pulling 2–3 real reference sites for the axis is even stronger seeding.

## Known AI-mode center (cap: one card per round)

- **Cream editorial** — warm cream bg (≈#F4F1EA), high-contrast serif display, terracotta accent.
- **Dark acid** — near-black bg, single acid-green or vermilion accent.
- **Broadsheet default** — hairline rules, zero border-radius, dense newspaper columns.
- **Generic SaaS bento** — even rounded-card grid, big number + small label + gradient accent.

## Archetype families (one-line signatures)

**Grid & order**
- **Swiss / International** — locked grid, flat, grotesk type, red/black/white restraint, zero decoration.
- **Bauhaus** — geometric primitives, primary colors, form-follows-function poster energy.
- **Editorial / magazine** — multi-column, drop caps, pull quotes, art-directed image crops.
- **Bento grid** — Apple-style modular tiles, each cell one idea, mixed cell sizes.
- **Swiss Modernism 2.0** — the grid loosened with oversized type and one expressive accent.

**Raw & loud**
- **Brutalism** — raw-HTML honesty, system fonts, harsh borders, no easing, no shadows.
- **Neubrutalism** — flat candy colors, thick black outlines, hard offset shadows, sticker energy.
- **Zine-punk** — photocopy grain, cut-out letters, torn edges, stapled-together collage.
- **Anti-design / raw** — deliberately unpolished, hand-drawn marks, exposed seams.
- **Memphis** — 80s squiggles, confetti geometry, clashing playful shapes.
- **Gen-Z maximalism** — layered stickers, chaotic type mixing, saturation without apology.

**Machine & instrument**
- **Terminal / CLI** — monospace everything, box-drawing characters, prompt-and-output rhythm.
- **HUD / sci-fi FUI** — thin strokes, corner brackets, radar arcs, telemetry overlays.
- **Cyberpunk** — neon on black, glitch, scanlines, chromatic aberration.
- **Braun / Rams industrial** — instrument-panel restraint, dial-and-switch logic, engraved labels.
- **Blueprint / schematic** — cyanotype linework, dimension annotations, exploded views.
- **Cartographic** — map conventions: contour lines, coordinates, legends, route markers.
- **Airport / transit signage** — wayfinding type, high-contrast color-coded panels, pictograms.

**Depth & material**
- **Glassmorphism** — frosted translucency, background blur, layered light.
- **Claymorphism** — soft inflated 3D forms, pastel clay, rounded everything.
- **Neumorphism** — surface-extruded soft shadows, monochrome tactility.
- **Skeuomorphism** — real materials: stitched leather, brushed metal, paper stacks.
- **Dimensional layering** — stacked planes, depth via parallax and cast shadows.
- **Aurora / gradient mesh** — ambient soft color fields, dreamy atmosphere.

**Era & nostalgia**
- **Y2K** — chrome type, bubbles, cyber-kitsch gradients, early-web optimism.
- **Vaporwave** — pastel neon, retro-web artifacts, ironic classicism.
- **Retro-futurism** — 60s space-age curves, atomic motifs, optimistic tomorrow.
- **Vintage analog / film** — grain, light leaks, faded stock, print registration errors.
- **Pixel art** — 8-bit sprites, hard edges, tile logic.
- **E-ink / paper** — book margins, one accent, distraction-free reading calm.

**Organic & calm**
- **Biophilic / organic** — plant motifs, curved growth lines, earth palette.
- **Wabi-sabi / quiet** — asymmetric, imperfect, muted, unhurried.
- **Nature distilled** — mineral neutrals, artisan restraint, matte textures.
- **Exaggerated minimalism** — very few elements at extreme scale, luxury whitespace.

**Data & function**
- **Tufte data-dense** — maximal data-ink, small multiples, sparklines, zero chrome.
- **Real-time monitoring** — status boards, alert states, live tickers, uptime greens.
- **Executive dashboard** — few large KPIs, calm hierarchy, decision-first.

**Type-led**
- **Poster / bold type** — one giant typographic statement, everything else subordinate.
- **Kinetic typography** — type in motion as the hero surface.
- **Academia** — scholarly serifs, footnotes, marginalia, numbered plates.

## Poles for finer axes (claim the extremes first — recipe §3.5)

- **Layout**: symmetric locked grid ↔ broken asymmetry (overlap, diagonals, oversized-vs-tight); hero-top ↔ hero-side ↔ no hero (uniform field).
- **Density**: one glanceable number ↔ full expert breakdown.
- **Data-viz**: bare numerals ↔ fully graphic encodings (tile matrix, rings, bars, dot grids, sparklines).
- **Color/texture**: monochrome + one accent ↔ polychrome subject-coded; flat ↔ atmospheric (grain, mesh, blur, shadow play).
- **Typography**: neutral utilitarian ↔ characterful display-led; one family ↔ display + body + mono trio. (When a typography round is truly open and tokens don't lock fonts, avoid the reflex faces — Inter/Roboto/Arial read as unstyled defaults.)

## Execution notes per card

- **Commit to the assigned language.** Indecisive middles read as stock templates; a half-applied exemplar reads as the mode with a costume.
- **Match execution to direction**: a maximalist card needs elaborate detail; a minimalist card needs precision in spacing and type. Half-committing either way reads as unfinished.
