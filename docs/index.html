别再许愿式改 UI · 一次出 5 个版本并排挑

45s 宣传片 · 真实项目实录

它怎么干活

1发牌

一次生成 5 个结构真不同的版本——真数据、真 HTML,不是同一版换五个配色。并排 serve 给你在浏览器里挑。

2锁定

你指哪版,就锁哪版。喜欢 A 的布局 + B 的配色?可以混。锁进账本的东西,后面每一轮都逐字保真。

3收敛

每轮只动一个变量,几轮就到「就是这版」——然后用真实技术栈落地,和你锁定的 mockup 做视觉 diff 精修。

30 秒装上
